# TRACERSData

[![Build Status](https://github.com/JuliaSpacePhysics/TRACERSData.jl/actions/workflows/CI.yml/badge.svg?branch=main)](https://github.com/JuliaSpacePhysics/TRACERSData.jl/actions/workflows/CI.yml?query=branch%3Amain)
[![Coverage](https://codecov.io/gh/JuliaSpacePhysics/TRACERSData.jl/branch/main/graph/badge.svg)](https://codecov.io/gh/JuliaSpacePhysics/TRACERSData.jl)

Load data from the TRACERS (Tandem Reconnection and Cusp Electrodynamics Reconnaissance Satellites) mission.

**Documentation**: [![Dev](https://img.shields.io/badge/docs-dev-blue.svg)](https://JuliaSpacePhysics.github.io/TRACERSData.jl/dev/)

**Agent skill**: at the terminal, run `npx skills add JuliaSpacePhysics/TRACERSData.jl`

References: [Website](https://tracers.physics.uiowa.edu/), [NASA Science](https://science.nasa.gov/mission/tracers/)
